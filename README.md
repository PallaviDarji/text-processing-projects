# Text Processing Projects

This repository contains three Python mini-projects:

## 1. Levenshtein Distance
Calculates the edit distance between two strings.

## 2. Sentence Cleaner
Cleans a sentence for search engine processing.

## 3. Spam Trigger Filter
Detects spam messages using keyword matching.

## 4. Sentiment Data Cleaning
Prepares noisy real-world text for sentiment analysis by removing symbols, hashtags, and stopwords.

## 5. Keyboard Typo Correction
Demonstrates why keyboard layout matters using weighted Levenshtein distance.

## Tools Used
- Python
- VS Code
